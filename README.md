## Install
```
$ npm install -g starbucks-cli
```

## Usage
```bash
$ starbucks --help

Usage:
  $ starbucks <address>          Show 5 nearst Starbucks with address
  $ starbucks <zipcode>          Show 5 nearst Starbucks with zipcode

Example:
  $ starbucks '1201 S Figueroa St'
  $ starbucks 'M6K 3P6'
```
